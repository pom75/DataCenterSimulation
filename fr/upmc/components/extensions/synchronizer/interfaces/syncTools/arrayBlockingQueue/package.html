Interfaces pour l'utilisation du ArrayBlockingQueue.

Le package fr.upmc.components.extensions.synchronizer.interfaces.syncTools.arrayBlockingQueue définit les interfaces nécessaires pour utiliser l'ArrayBlockingQueue (fr.upmc.components.extensions.synchronizer.components.syncTools.ArrayBlockingQueue). Celui-ci reprend les fonctionnalités de java.util.concurrent.ArrayBlockingQueue. Il y a deux services pour cet outil :

  • Gett : utilisé pour retirer un objet au tampon.
  • Put : utilisé pour ajouter un objet au tampon.
