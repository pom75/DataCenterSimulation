Ports d'accès aux services du CountDownLatch

Le package fr.upmc.components.extensions.synchronizer.ports.syncTools.countDownLatch fournit les ports utiles pour l'outil de synchronisation CountDownLatch.

On dispose de :

  • Ports inbound/outbound pour le service de connexion (création d'un nouveau port client)
  • Ports inbound/outbound pour le service de synchronisation
