Ports d'accès aux services du ArrayBlockingQueue

Le package fr.upmc.components.extensions.synchronizer.ports.syncTools.arrayBlockingQueue fournit les ports utiles pour l'outil de synchronisation ArrayBlockingQueue.

On dispose de :

  • Ports inbound/outbound pour le service de connexion (création d'un nouveau port client)
  • Ports inbound/outbound pour le service TAKE (consommateur)
  • Ports inbound/outbound pour le service PUT (producteur)
