Ports d'accès aux services du CyclicBarrier

Le package fr.upmc.components.extensions.synchronizer.ports.syncTools.cyclicBarrier fournit les ports utiles pour l'outil de synchronisation CyclicBarrier.

On dispose de :

  • Ports inbound/outbound pour le service de connexion (création d'un nouveau port client)
  • Ports inbound/outbound pour le service de synchronisation
