Ports d'accès aux services du Semaphore

Le package fr.upmc.components.extensions.synchronizer.ports.syncTools.semaphore fournit les ports utiles pour l'outil de synchronisation Semaphore.

On dispose de :

  • Ports inbound/outbound pour le service de connexion (création d'un nouveau port client)
  • Ports inbound/outbound pour le service de synchronisation
