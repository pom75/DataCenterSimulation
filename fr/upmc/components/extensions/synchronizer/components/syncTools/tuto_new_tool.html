Création d'un nouveau composant de synchronisation

  1. Mettre en place le nouveau service :
    • Créer le composant (hérite de AbstractComponent) avec une méthode getOwnPortURI (1 fichier)
    • Créer les deux interfaces et les deux ports inbound et outbound pour le service de création de port dynamique + le connecteur (5 fichiers)
    • Créer les deux interfaces et les deux ports inbound et outbound pour le premier service de synchronisation + le connecteur (5 fichiers)
    • Faire pareil que précédemment pour les éventuels autres services ...
  2. Faire les ajouts dans le SynchronizerManager :
    • Ajouter les 3 champs spécifiques et les initialiser dans le constructeur
    • Ajouter la méthode provide[nouvel_outil]URI sur le modèle des autres méthodes provide; /!\ Attention : Les paramètres des constructeurs des composants de synchronisation doivent tous hériter de Object (pas de type primitif)
    • La nouvelle méthode provide étant un nouveau service du composant SynchronizerManager, il faut propager la mise à jour dans tous les services, ports et connecteur du SynchronizerManager
    • Rajouter une méthode statique encode[outil]ID qui prend en paramètre un ID, et tous les paramètres du constructeur de l'outil (Ces paramètres doivent pouvoir être mis sous forme de string). Cette méthode sera utilisée par l'assembly pour créer des ID de groupe pour ce type d'outil.
  3. Faire les ajouts dans le délégué :
    • Ajouter un nouveau if dans la méthode getPort correspondant aux nouveaux services précédemment mis en place.
    • Ce if doit lancer l'appel à la méthode connect[outil] qu'il faudra ajouter sur le modèle des autres méthodes de ce genre.
