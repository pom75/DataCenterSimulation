Interfaces and data for sensor services

The package fr.upmc.components.examples.smoothing.sensor.interfaces provides two interfaces for sensor services, a data offered interface SensorDataOfferedI for the sensor service providers and SensorDataRequiredI for the sensor service clients. The two interfaces define an interface that must be implemented by a data carrying object. The class SensorData provide an implementation of both interfaces for sensor data that are simply (double) real numbers.
