Interfaces for random number generation services

The package fr.upmc.components.examples.smoothing.rng.interfaces provides interfaces for random number generation components. The component interfaces RNGDataOfferedI and RNGDataRequiredI.java are the ones used by components to offer and require the service respectively. The component interface RNGTriggerI defines the complementary which controls the generation in push mode, with methods to start and stop the generation.

The Java interface RNGProducerI defines the service implementation methods common to all random number generation components.
