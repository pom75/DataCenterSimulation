Components for the dynamic client/server example

The dynamic client/server example uses three components:

  • DynamicURIConsumer is the dynamic version of the component URIConsumer of the basic client/server example. It merely adds to the later the dynamic connection service of the interface DynamicallyConnectableComponentI.
  • DynamicURIProvider is the dynamic version of the component URIProvider of the basic client/server example. It also adds to the later the dynamic connection service of the interface DynamicallyConnectableComponentI, but it would not really be needed for such client/server connections since we only need to connect on the client side in this case.
  • DynamicAssembler that creates the other components, makes them interconnect, and starts them.
